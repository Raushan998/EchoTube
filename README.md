## Requirements

Please check the Gemfile to ruby and rails version.

## Installation

You should have below installed requirements
* [ruby](https://rvm.io/)
* [rails](https://rvm.io/)
* [postgres](https://www.postgresql.org/)
* [node](https://nodejs.org/en)

```
bundle install
```

```
npm install
```

## Usage

```
./bin/dev
```
